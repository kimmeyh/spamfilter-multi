Runs the rules engine against the current inbox. The status banner shows total emails, deletions, moves, safe senders, and errors. Progress updates are throttled for performance -- final counts always appear when the scan completes.

Scan mode is chosen in Settings; the default is read-only (dry run). Switch to an action mode to actually delete or move matched emails. The Select Account icon returns to the account list; View Scan History shows past scans for this account.
